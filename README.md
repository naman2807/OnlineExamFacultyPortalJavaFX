# Online Exam Faculty Portal JavaFX
This is the GUI based application for faculty to conduct online exam.
# Functionality
1. Login
2. Start Exam
3. Add Exam Questions
4. Add Viva Questions
5. Display All Exam Questions
6. Display All Viva Questions
7. End Exam
